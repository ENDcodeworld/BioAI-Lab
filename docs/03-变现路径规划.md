## 附录：定价计算器

```python
# 简化版收入计算器
def calculate_revenue(saas_users, course_students, consulting_deals):
    saas_revenue = saas_users * 1999  # 专业版月费
    course_revenue = course_students * 4999  # 课程单价
    consulting_revenue = consulting_deals * 80000  # 咨询均价
    
    total = saas_revenue + course_revenue + consulting_revenue
    return {
        'saas': saas_revenue,
        'course': course_revenue,
        'consulting': consulting_revenue,
        'total': total
    }

# 示例：第 6 个月目标
print(calculate_revenue(saas_users=40, course_students=30, consulting_deals=2))
# 输出：{'saas': 79960, 'course': 149970, 'consulting': 160000, 'total': 389930}
```
