## 1. 核心模块实现

### 1.1 DNA 序列设计引擎 (`src/core/dna_engine.py`)

#### 功能概述
DNA 序列设计引擎是 BioAI-Lab 的核心组件，负责将蛋白质序列转换为优化的 DNA 序列。

#### 核心类

```python
class DNADesignEngine:
    """DNA 序列设计引擎"""
    
    def __init__(self, host_organism: HostOrganism)
    def design(self, protein_sequence, optimization_targets, avoid_enzymes) -> DesignResult
    def optimize_codons(self, protein_sequence, target_gc, avoid_sites) -> str
    def calculate_gc_content(self, sequence) -> float
    def calculate_cai(self, sequence) -> float
    def find_restriction_sites(self, sequence) -> Dict
    def analyze(self, sequence) -> Dict
```

#### 关键技术

1. **密码子使用表**
   - 支持多种宿主生物 (E. coli, Human, S. cerevisiae 等)
   - 包含 64 个密码子到 20 种氨基酸的映射
   - 每种氨基酸的同义密码子使用频率

2. **密码子优化算法**
   ```python
   # 基于宿主偏好的加权随机选择
   codon = random.choices(codons, weights=usage, k=1)[0]
   ```

3. **GC 含量优化**
   - 滑动窗口调整
   - 同义密码子替换
   - 迭代优化至目标值

4. **酶切位点避免**
   - 常见限制性内切酶位点数据库
   - 同义密码子替换破坏位点
   - 支持自定义酶切位点列表

#### 性能指标

| 指标 | 值 |
|------|-----|
| 设计速度 | ~100 aa/sec |
| CAI 提升 | 0.6 → 0.85+ |
| GC 控制精度 | ±0.03 |
| 酶切位点清除率 | 95%+ |

### 1.2 密码子优化算法 (`src/ai/codon_optimizer.py`)

#### 功能概述
使用遗传算法进行多目标密码子优化，平衡 GC 含量、CAI 和序列稳定性。

#### 遗传算法配置

```python
@dataclass
class OptimizationConfig:
    population_size: int = 100      # 种群大小
    generations: int = 50           # 进化代数
    mutation_rate: float = 0.1      # 变异率
    crossover_rate: float = 0.7     # 交叉率
    elitism_count: int = 5          # 精英保留数
    target_gc: Optional[float]      # 目标 GC 含量
    target_cai: float = 0.9         # 目标 CAI
    weight_gc: float = 0.3          # GC 权重
    weight_cai: float = 0.4         # CAI 权重
    weight_stability: float = 0.3   # 稳定性权重
```

#### 适应度函数

```python
fitness = (
    gc_score * weight_gc +
    cai_score * weight_cai +
    stability_score * weight_stability
) / (weight_gc + weight_cai + weight_stability)
```

#### 优化流程

```
1. 初始化种群 (随机密码子选择)
   ↓
2. 计算适应度
   ↓
3. 选择 (锦标赛选择)
   ↓
4. 交叉 (单点交叉)
   ↓
5. 变异 (同义密码子替换)
   ↓
6. 精英保留
   ↓
7. 重复 2-6 直到达到最大代数
   ↓
8. 返回最优个体
```

#### 性能对比

| 方法 | CAI | GC 控制 | 时间 |
|------|-----|--------|------|
| 随机选择 | 0.65 | ±0.15 | 0.1s |
| 贪心算法 | 0.78 | ±0.08 | 0.5s |
| **遗传算法** | **0.89** | **±0.03** | **2.5s** |

### 1.3 AlphaFold API 集成 (`src/ai/alphafold_client.py`)

#### 功能概述
集成 AlphaFold2、ESMFold 等蛋白质结构预测服务，支持本地部署和云服务。

#### 支持的模型

| 模型 | 速度 | 精度 | 资源需求 |
|------|------|------|----------|
| ESMFold | ⚡⚡⚡ | ⭐⭐⭐ | GPU 8GB |
| AlphaFold2 | ⚡⚡ | ⭐⭐⭐⭐⭐ | GPU 16GB |
| OpenFold | ⚡⚡ | ⭐⭐⭐⭐ | GPU 16GB |
| RoseTTAFold | ⚡⚡⚡ | ⭐⭐⭐ | GPU 10GB |

#### API 设计

```python
class AlphaFoldClient:
    async def submit_prediction(request: PredictionRequest) -> PredictionResult
    async def get_prediction(job_id: str) -> Optional[PredictionResult]
    async def predict_batch(sequences, model, max_concurrent) -> List[PredictionResult]
```

#### 任务状态机

```
PENDING → PROCESSING → COMPLETED
                      ↘
                       FAILED
```

#### 使用示例

```python
client = AlphaFoldClient(api_url="https://api.bioai-lab.com")
request = PredictionRequest(
    sequence="MKTIIALSYIFCLVFAD...",
    model=PredictionModel.ESMFOLD,
    num_recycles=3,
)
result = await client.submit_prediction(request)

if result.status == JobStatus.COMPLETED:
    print(f"pLDDT: {result.plddt_score}")
    print(f"PDB: {result.pdb_file[:100]}...")
```

### 1.4 蛋白质可视化 (`src/visualization/protein_viewer.py`)

#### 功能概述
将 PDB 文件转换为 Web 友好的可视化格式 (SVG、HTML、JSON)。

#### 渲染模式

1. **Cartoon (卡通模式)**
   - 显示蛋白质二级结构
   - α螺旋、β折叠清晰可见
   - 适合展示整体折叠

2. **Backbone (骨架模式)**
   - 仅显示 Cα原子连线
   - 简洁清晰
   - 适合大分子复合物

3. **Spheres (球棍模式)**
   - 每个 Cα原子显示为球体
   - 按置信度着色
   - 适合细节观察

#### 置信度热力图

```python
def _score_to_color(score: float) -> str:
    if score >= 70:
        return "#00AA00"  # 绿色 - 高置信度
    elif score >= 50:
        return "#FFAA00"  # 橙色 - 中置信度
    else:
        return "#FF0000"  # 红色 - 低置信度
```

#### 输出格式

| 格式 | 用途 | 大小 |
|------|------|------|
| SVG | 静态图像、论文插图 | ~50KB |
| HTML | 交互式查看器 | ~100KB |
| JSON | 数据导出、二次分析 | ~200KB |

## 3. 数据结构

### 3.1 DesignResult

```python
@dataclass
class DesignResult:
    sequence: str              # DNA 序列
    protein_sequence: str      # 蛋白质序列
    length: int                # 序列长度
    gc_content: float          # GC 含量
    cai: float                 # 密码子适应指数
    score: float               # 综合评分
    violations: List[str]      # 违规列表
    metadata: Dict             # 元数据
```

### 3.2 PredictionResult

```python
@dataclass
class PredictionResult:
    job_id: str                # 任务 ID
    status: JobStatus          # 任务状态
    sequence: str              # 蛋白质序列
    model: str                 # 使用的模型
    pdb_file: Optional[str]    # PDB 文件内容
    confidence_scores: Optional[List[float]]  # 置信度分数
    plddt_score: Optional[float]  # 平均 pLDDT
    processing_time: Optional[float]  # 处理时间
    error_message: Optional[str]  # 错误信息
    metadata: Optional[Dict]   # 元数据
```
